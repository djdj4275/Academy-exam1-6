git 다운 / git hub 들어가서 회원가입

$ git config --global user.name "John Doe"
$ git config --global user.email johndoe@example.com

$ git init (초기화)
$ git push -u origin master (main이 아니라 master로 수정)
$ git remote set-url origin https://github.com/djdj4275/gitTest.git
$ git log

commit = 저장 
remote = 보내기
